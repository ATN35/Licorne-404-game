# Licorne 404 Game

Un mini-jeu d'arcade où vous incarnez une licorne poursuivie par l'erreur 404 !

## Jouer en ligne

Accédez au jeu ici :

➡️ https://licorne-404-game.vercel.app/

## Description rapide
- Contrôlez la licorne à la souris
- Évitez le 404, les missiles, les flammes et la scie
- Récupérez les bonus pour survivre plus longtemps

Bon jeu ! 🦄
